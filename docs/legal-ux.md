# LAW.OS — Legal UX (변호사법 대응)

> **Version**: 1.0
> **Priority**: 🔴 Critical — 이 문서의 규칙을 따르지 않으면 **변호사법 제109조** 위반 소지가 있습니다.
> **Disclaimer**: 이 문서는 법률 자문이 아니며, 프로덕션 출시 전 반드시 변호사 검토를 받아야 합니다.

---

## 배경

### 변호사법 제109조 (요약)
> 변호사가 아니면서 금품·향응 기타 이익을 받거나 받을 것을 약속하고 법률사건에 관하여 **감정·대리·중재·화해·청탁·법률상담 또는 법률관계 문서 작성** 등 법률사무를 취급하는 행위를 금지한다.

### 핵심 논점
- LAW.OS는 **학습 도구(educational tool)**로 포지셔닝 → "법률 상담"이 아님
- 사용자의 **구체적 사실관계에 대한 법적 판단**을 제공하면 위반 소지
- **일반적 법률 정보 제공**은 허용 (교과서, 학술 자료와 동일)

---

## 제품 원칙 (반드시 준수)

### ✅ 해야 할 것
- 일반적인 법리·조문·판례 설명
- 학문적 해석
- 유사 판례 소개
- 학습 자료 큐레이션
- "이와 유사한 사안에서 대법원은 ~했습니다" 형식

### ❌ 하지 말아야 할 것
- **개인 사건 맞춤 조언** ("당신의 경우 승소할 수 있습니다")
- **소송 전략 제시** ("이 방법으로 소송하세요")
- **법률 문서 작성** (소장, 계약서, 합의서 드래프트)
- **결과 예측** ("이 사건은 승소 확률 80%입니다")
- **특정 변호사 추천**

## 런타임 면책 (In-App Disclaimers)

### 1. 채팅 화면 — 상시 하단 고지
```
┌─────────────────────────────┐
│ [AI 답변 영역]                │
│                              │
│ ─────────────────────────    │
│ ℹ️ 학습 참고용 · 실제 분쟁은   │
│   변호사 상담 필요 · 출처 확인 │
└─────────────────────────────┘
```
- 고정 배너, 스크롤에 영향받지 않음
- 탭 시 상세 고지 모달 (최초 동의 화면 3번 재표시)

### 2. 각 답변 하단 — Citation Footer
```
📚 출처: 민법 제750조, 대법원 2018다12345
⚠️ 이 답변은 학습 참고용입니다.
    인용된 조문/판례를 직접 확인하세요.
```

### 3. "개인 사건 조언 요청" 감지 시
사용자 질문이 특정 패턴을 포함하면 **차단 응답** 표시:
- "제가 ~했는데 승소할 수 있나요?"
- "이 경우 어떻게 소송해야 하나요?"
- "제 남편이 ~했는데..."
- "저는 ~입니다. 조언해주세요"

#### 차단 응답 템플릿
```
죄송합니다. LAW.OS는 법학 학습 도구로,
구체적인 개인 사건에 대한 법률 상담은 제공할 수 없습니다.

대신 다음을 도와드릴 수 있습니다:
• 관련 법리와 조문 설명
• 유사 판례의 일반적 법적 논리 소개
• 학습 자료 큐레이션

실제 법률 분쟁이 있으시다면 반드시 변호사와 상담하세요.
대한변호사협회 법률상담: 02-3476-4000
```

## 키워드 감지 시스템

### 차단 키워드 (RegExp)
```ts
const personalAdvicePatterns = [
  /제가|내가.*(했|하였)/,
  /(승소|패소).*(가능|확률|할 수 있)/,
  /(소송|고소).*(해야|하는 게 좋)/,
  /저(희)?(의)? (남편|아내|부모|자녀|친구|직장)/,
  /(해결|조치).*방법.*(알려|가르쳐)/,
  /계약서.*(작성|검토).*해/,
  /합의서.*(써|작성)/,
  /(고소장|소장).*(쓰|작성)/,
];
```

### 3단계 처리
1. **클라이언트 측 프리체크** → 경고 UI (부드러운 안내)
2. **서버 측 재체크** → 차단 응답 강제
3. **LLM 프롬프트 가드레일** → "개인 조언 요청 시 거부" 지시

## 법률 상담이 아닌 학습 — 구분 예시

### ❌ 개인 조언 (차단)
> Q: "제가 교통사고를 당했는데 보험사가 합의금 500만원을 제시했습니다. 소송하면 더 받을 수 있을까요?"
>
> 차단 응답 → 관련 법리 설명으로 유도 or 변호사 추천

### ✅ 학습 질의 (허용)
> Q: "교통사고 손해배상 청구 시 고려되는 요소는 무엇인가요?"
>
> 응답: 민법 제750조 + 자동차손해배상보장법 + 관련 판례 + 일반적 산정 기준

### 경계선 사례
> Q: "만약 교통사고 가해자가 도주하면 어떻게 되나요?"
>
> → 일반적 법리 설명 OK, "만약"이 붙었으므로 가설적 질문으로 간주

## 서재 (Notes) 저장 시 주의

- 원본 그대로 저장 (서비스가 재생성하지 않음)
- "이 내용은 법률 상담이 아닙니다" 워터마크 PDF 내보내기
- 공유 시 동일 고지 자동 첨부

## Deep Debate 특별 규정

Deep Debate 모드는 4명 에이전트가 법적 논거를 주고받아 **법률 상담처럼 보일 위험**이 큼. 추가 보호:

- Debate 시작 전 **별도 고지 모달**: "이 토론은 학습 목적의 가상 토론입니다"
- Judge 에이전트는 **결론에서 항상** "이는 학문적 견해이며 개별 사건은 다를 수 있습니다" 명시
- Narrator 요약 말미에 "실제 분쟁은 변호사와 상담하세요" 자동 삽입

## 데이터 보관 정책

### 민감 정보 수집 금지
- 실명 외 **주민등록번호, 전화번호, 주소 수집 금지**
- 사용자가 실수로 입력 시 → LLM 측에서 마스킹 또는 거부

### 채팅 기록 보관
- 개인 학습 기록이므로 사용자 자신만 열람 가능 (RLS)
- 운영 측은 **익명화 통계**만 집계
- 탈퇴 시 즉시 soft delete → 30일 후 hard delete
