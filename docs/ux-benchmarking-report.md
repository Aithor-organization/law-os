# Law-OS UX 벤치마킹 리포트

> 작성일: 2026-04-21
> 목적: 유명 앱 10개의 UX 장점을 law-os(법학 학습 앱)에 차용하여 종합 UX/UI 점수를 85점 이상으로 끌어올리기 위한 전략 문서

---

## 0. 법-OS 현황 요약

**앱 정체성**
- 사용자: 한국 로스쿨 / 변호사시험 준비 / 법학과 학습자
- 핵심 탭 4개: **대화**(AI 법률 Q&A) · **검색**(조문+판례) · **서재**(북마크) · **프로필**(통계)
- 디자인 톤: 다크 배경 + 바이올렛 accent(#A855F7) + 모노 + 한글 혼합 "Dark Academia Pro"
- 스택: Expo React Native + NativeWind 4 + Supabase

**현재 점수 추정치**: 68/100 (UX 41/60, UI 27/40) — "개선 필요" 구간

**주요 페인 포인트**
- 탭 간 인터랙션 패턴 드리프트 (SegmentedTabs vs 카테고리 카드)
- 프로필 통계 평탄한 위계
- 이모지와 모노 텍스트 혼용
- 로딩/빈 상태가 얕음 (스켈레톤은 있으나 실제 콘텐츠 모양 미반영)

## 1. 조사 대상 앱 (10개)

law-os의 4가지 성격(AI 대화 / 지식 검색 / 컬렉션 / 학습)을 커버하도록 선정.

### 1. Perplexity
- **카테고리**: AI 검색·답변
- **주요 패턴**:
  - **팔로우업 질문 버튼** — 답변 하단에 "이어서 물어볼 질문" 3-4개 제안 버튼
    - 왜: 타이핑 진입장벽 제거, 대화 지속성 ↑
    - 구현: `<View flexDirection="row" flexWrap="wrap">` 안에 Pressable 칩
    - **law-os 관련성: HIGH → 대화 탭**
  - **Sources 섹션 (출처 투명성)** — 답변 아래 참고 출처 링크를 시각적 카드로 표시
    - 왜: 법률 분야 특히 중요한 신뢰도 증거
    - 구현: `<SourceCard url={...} title={...} />` 컴포넌트
    - **law-os 관련성: HIGH → 검색 탭**
  - **음성 입력 플로팅 버튼** — 하단 중앙 마이크, 한 손 접근
    - **law-os 관련성: MED → 대화 탭**

### 2. ChatGPT (iOS)
- **카테고리**: AI 대화
- **주요 패턴**:
  - **모델 선택 상단 드롭다운** — GPT-4/4o 등 전환을 상단에서 1탭
    - **law-os 관련성: MED → 대화 탭** ("초보자 모드" vs "전문가 모드" 전환 대안)
  - **좌측 드로어 대화 히스토리** — 슬라이드 메뉴로 과거 대화 접근
    - 왜: 모바일 세로 공간 절약, 한 손 탐색
    - **law-os 관련성: HIGH → 대화 탭**
  - **동적 높이 입력창** — 다중 라인 입력 시 창이 커지되 대화 가려지지 않음
    - **law-os 관련성: HIGH → 대화 탭**

### 3. Notion (Mobile)
- **카테고리**: 지식 노트·데이터베이스
- **주요 패턴**:
  - **플로팅 '+' 액션 버튼** — 하단 우측에서 신규 생성 1탭
    - **law-os 관련성: HIGH → 서재 탭** (빠른 북마크 클리핑)
  - **토글 헤더 (접기/펴기)** — 섹션별 접을 수 있어 정보 과부하 방지
    - **law-os 관련성: HIGH → 프로필 탭** (통계/배지/학습기록 섹션 구조화)
  - **상단 고정 검색창** — 전역 검색 1단계 접근
    - **law-os 관련성: HIGH → 검색 탭**

### 4. Apple Books
- **카테고리**: 전자책 리더
- **주요 패턴**:
  - **우측 프로그레스 바** — 페이지 진도를 화면 우측 컬러 바로 시각화
    - 왜: 심리적 만족감, 남은 분량 한눈
    - **law-os 관련성: HIGH → 학습 탭** (법학 교과서 진도 표시)
  - **텍스트 선택 컨텍스트 메뉴 — 하이라이트** — 선택 → 색상+메모 한 번에
    - **law-os 관련성: HIGH → 서재 탭** (법령 조항/판례 핵심 하이라이트)
  - **밤 모드 자동 전환**
    - **law-os 관련성: MED → 전역 설정**

### 5. Readwise
- **카테고리**: 하이라이트 복습·스마트 큐
- **주요 패턴**:
  - **Daily Review 푸시** — 매일 아침 지난 하이라이트 카드 스택 복습 세션
    - 왜: Spaced repetition, 자동 스케줄링
    - **law-os 관련성: HIGH → 전역 + 프로필 탭** ("오늘의 법령" / "복습 대상 판례")
  - **출처 뱃지** — 각 하이라이트가 어느 책/판례집에서 왔는지
    - **law-os 관련성: HIGH → 서재 탭** (저장한 법령/판례의 출처 판례집 명시)
  - **Reading List 큐** — "읽을 목록 (3)" 같은 명확한 다음 단계
    - **law-os 관련성: MED → 서재 탭**

### 6. Duolingo
- **카테고리**: 학습 게이미피케이션
- **주요 패턴**:
  - **스트릭 카운터 🔥** — 연속 학습일을 불 아이콘 + 숫자로 상단 고정
    - 왜: 손실 회피(loss aversion) 심리, 연속성 유지 동기
    - **law-os 관련성: HIGH → 프로필 탭** ("법학 공부 연속 일수")
  - **레슨 진행 바** — 한 유닛 안에서 얼마나 남았는지
    - **law-os 관련성: HIGH → 대화 탭** (주제별 학습 진도)
  - **목표 달성 축하 (Lottie + Haptics)** — 즉시 보상
    - **law-os 관련성: HIGH → 전역** (일일 목표 달성 축하 모달)

### 7. Toss
- **카테고리**: 한국 핀테크
- **주요 패턴**:
  - **카드 모핑 애니메이션** — 리스트 → 상세로 부드럽게 확대
    - **law-os 관련성: MED → 검색/서재 탭**
  - **날짜별 스티키 헤더** — 거래를 날짜로 자동 그룹화
    - 왜: 정보 계층 명확, 특정 기간 스캔 용이
    - **law-os 관련성: HIGH → 검색 탭** (검색 결과 카테고리 그룹화)
  - **자주 쓰는 송금 대상 Quick Send** — 반복 행동 진입장벽 극소화
    - **law-os 관련성: MED → 대화 탭** ("자주 물은 주제" 즐겨찾기)

### 8. KakaoTalk Wallet
- **카테고리**: 한국 메시징 + 금융
- **주요 패턴**:
  - **바텀 탭 + 아이콘 + 한글 레이블** — 한국 사용자 기대치 표준
    - **law-os 관련성: HIGH → 전역 네비게이션** (현재 mono 영문 탭 레이블 교체 검토)
  - **탭 뱃지 (미읽 카운터)** — 빨간 뱃지로 주의 유도
    - **law-os 관련성: HIGH → 전역 네비게이션** ("복습 대상: 3" 같은 뱃지)
  - **상단 고정 요약 카드** — 현황을 항상 시각화
    - **law-os 관련성: HIGH → 프로필 탭** ("오늘 학습 2시간 · 이번주 문제 15")

### 9. Linear (Mobile)
- **카테고리**: 협업 작업 관리
- **주요 패턴**:
  - **낙관적 업데이트 (Optimistic UI)** — 버튼 누르면 서버 응답 기다리지 않고 즉시 반영
    - 왜: 앱 반응성 체감 향상, 네트워크 지연 마스킹
    - **law-os 관련성: HIGH → 대화/서재 탭** (북마크 토글, 필터 변경)
  - **인라인 완료 체크** — 체크박스 탭으로 줄긋기 처리, 상세 이동 없음
    - **law-os 관련성: HIGH → 서재 탭** ("복습 완료" 인라인 체크)
  - **실시간 협업 표시** — 다른 사용자 위치 시각화
    - **law-os 관련성: LOW** (law-os는 단일 사용자)

### 10. Things 3
- **카테고리**: 개인 작업 관리 (애플 디자인상)
- **주요 패턴**:
  - **Quick Entry** — 어디서든 빠른 입력창 팝업
    - **law-os 관련성: HIGH → 대화 탭** (생각난 질문 즉시 캡처)
  - **Today/Upcoming/Later 시간대 그룹화** — 자동 분류 + 스티키 헤더
    - **law-os 관련성: HIGH → 프로필 탭** ("이번주 목표" / "다음달 도전")
  - **스무스한 전환 애니메이션** — 항목 완료/삭제 시 spring 애니메이션
    - **law-os 관련성: MED → 전역**

## 4. 구현 우선순위 (85점 달성 로드맵)

**Phase α (즉시, 1-2 세션)** — 점수 영향 큰 저비용 개선
- #9 Sources 인용 카드 → 검색 탭 (V1 위계 + U2 피드백 향상)
- #2 탭 뱃지 → 전역 (U1 네비 명확성 향상)
- #6 스티키 헤더 그룹화 → 검색 탭 (U4 정보 계층)
- 이전 리뷰의 **이모지 → SVG 모노 아이콘 교체** (V4 +3점)

**Phase β (3-5 세션)** — 중간 투자, 큰 ROI
- #1 팔로우업 질문 버튼 → 대화 탭
- #3 스트릭 카운터 → 프로필 탭 (바이올렛 불 아이콘으로 톤 매치)
- #4 토글 헤더 → 프로필 탭 (U4 위계 개선 +3점)
- #5 낙관적 업데이트 → 대화/서재

**Phase γ (6+ 세션)** — 깊이 있는 UX 향상
- #7 빠른 입력 FAB
- #8 카드 모핑 전환
- #10 학습 진도 바
- Daily Review 푸시 + spaced repetition
